Effektivt Samarbeid på GitHub

Ved bruk av GitHub har vi vært i stand til å administrere prosjektet vårt med letthet og effektivitet. Vi brukte separate grener for utvikling, slik at alle kunne jobbe selvstendig uten å påvirke hovedgrenen før koden ble gjennomgått og godkjent.

Regelmessige commits har bidratt til å holde arbeidet oversiktlig og gjør det lett å spore hver enkeltes bidrag.

Fordeler med Git og GitHub

  • Versjonskontroll gjør det lett å holde oversikt over alle endringer.
  • Samarbeid i sanntid med hver person på sitt eget arbeid.
  • Mulighet for sporing og håndtering av eventuelle kodekonflikter.
